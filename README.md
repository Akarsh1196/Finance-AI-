# 💰 Finance AI – Intelligent Personal Finance System

🚀 Live Demo: https://personal-finance-t.streamlit.app

---

## 🧠 Overview

Finance AI is an AI-powered personal finance platform that analyzes user behavior, predicts financial risk, and provides goal-based investment recommendations.

It combines Machine Learning + FinTech concepts to simulate real-world financial decision-making.

---

## ✨ Features

- 🔐 Google OAuth Login (Secure Authentication)
- 🧠 Financial Behavior Analysis (Clustering)
- ⚠️ Risk Assessment Model
- 💰 ML-based Investment Prediction
- 🎯 Goal-based SIP Calculator
- 📊 Interactive Dashboard (Streamlit)
- 💡 Smart Financial Recommendations

---

## 🛠 Tech Stack

- **Frontend:** Streamlit  
- **Backend:** Python  
- **ML Models:** Scikit-learn  
- **Data Processing:** Pandas, NumPy  
- **Auth:** Google OAuth  
- **Deployment:** Streamlit Cloud  

---

## 🧪 Machine Learning Used

- KMeans → Financial Behavior Clustering  
- Regression → Investment Prediction  
- Rule-based + ML Hybrid → Recommendations  

---

## 🎯 Problem It Solves

Most people:
- Don’t understand their financial behavior  
- Invest randomly  
- Lack goal-based planning  

👉 This system helps users:
- Understand their finances  
- Reduce risk  
- Plan smarter investments
